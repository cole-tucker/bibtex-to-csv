Frequently Asked Questions

How do I run this application?
To run this application you'll need a BibTeX file in advance. Click on the "Choose File" button and select the file from your computer. Afterwards choose wether you would like to have the output file sorted by author or by year. Once that is done click the "Generate CSV" button and a popup will confirm that the CSV file has been downloaded to your computer.
